# Boids

鳥の群れや魚の群れの動きをシミュレートする人工生命モデル。

## 3つの基本ルール

|ルール|概要|
|---|---|
|分離|近くの仲間と衝突をするのを避けるため、近づきすぎた仲間から遠ざかる方向に舵を切る|
|整列|群れ全体の進行方向を合わせるため、近傍の仲間の向きと速さの平均に刃部を合わせる|
|結合|群れから孤立しないように、近傍の仲間の重心(平均位置)に向かって引き寄せられる|

## 実装の流れ

1. CPU
    - ウィンドウ作成とグラフィックス準備
    - ボイドの初期位置や速度のデータをGPUバッファにアップロード
    - 計算用と描画用の2つのパイプラインを準備
2. GPU
    - 3つのルールを適用して、次のフレームのボイドの位置と速度を計算・更新
    - 三角形などのシンプルな形状を、更新された位置にバッファを基に大量に画面に描画
