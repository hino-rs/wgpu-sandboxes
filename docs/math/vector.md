# ベクトル

## 演算

### 加減

#### 足し算

移動のハシゴ

$$[2, 3] + [4, -1] = [6, 2]$$

> 右に2歩、上に3歩進め
> +右に4歩、下に1歩進め

#### 引き算

ターゲットへの矢印を作る

> 目的地の位置 - 現在の位置 = 目的地への矢印

- 自分の位置: [2, 3]
- 敵の位置:   [6, 5]

$$[6, 5] - [2, 3] = [4, 2]$$

敵へのベクトル: [4, 2]

### 乗算

#### 内積 (Dot Product): どれくらい同じ方向を向いているか？

内積の結果は矢印ではなくただ1つの数値になる。

|結果|意味|ライティングの例|
|---|---|---|
|結果がプラス|2つの矢印が**だいたい同じ方向**を向いている|正面衝突で一番明るくなる|
|結果がゼロ|2つの矢印が**完全に真横(90度垂直)**を向いている|光が当たらない|
|結果がマイナス|2つの矢印が**お互い逆の方向**を向いている|影(真っ暗)になる|

#### 外積 (Cross Product): 2つの矢印に垂直な、第3の矢印を作る

内積と違って、外積では新たな矢印が作られる。
右手の法則のように、親指、人差し指という2つのベクトルA,Bをもとに中指と言うベクトルCが生まれる。

カメラに向かって表を向いているポリゴンは描画し、裏を向いているポリゴンは描画をスキップするカリングと言う技術に役立つ。

### 演算問題

#### 第1問

> あなたの作ったゲームのキャラクターが、現在マップの $[2, 1]$ の位置に立っています。
プレイヤーがコントローラーのボタンを押して、キャラクターに $[3, 5]$ という「移動ベクトル」を与えました。

- 問題： 移動した後の、キャラクターの新しい位置（座標）はどこになるでしょうか？

$$[2, 1] + [3, 5] = [5, 6]$$

#### 第2問

- 自分の飛行機（プレイヤー）の位置： $[1, 2]$
- 狙うべき敵（ボス）の位置： $[5, 5]$
- 問題： プレイヤーから敵に向かってまっすぐレーザーを撃ちたいです。レーザーを飛ばすべき「向きと大きさのベクトル（敵への矢印）」はいくつになるでしょうか？

$$[5, 5] - [1, 2] = [4, 3]$$

#### 第3問

> ステルスゲーム（隠れるゲーム）を作っています。敵の視線が「真右」を向いているとします。これをベクトルで表すと $[1, 0]$ です。プレイヤーが、敵の真上から忍び寄ろうとしています。このプレイヤーの接近ベクトルを $[0, 1]$（真上）とします。

- 問題： 敵の視線ベクトル $[1, 0]$ と、プレイヤーの接近ベクトル $[0, 1]$ の内積を計算してみてください。
- 計算結果から、プレイヤーは敵に見つかりそう（同じ向き）ですか？ それとも死角（垂直）にいますか？

**内積の計算式**:

$$a ⋅ b = a1 ​× b1 ​+ a2​ × b2​$$

**実際に計算**:

$$[1, 0] ⋅ [0, 1] = (1 * 0) + (0 * 1) = 0$$

= 垂直に死角
